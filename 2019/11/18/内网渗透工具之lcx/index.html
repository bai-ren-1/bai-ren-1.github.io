内网渗透工具总结之lcx

因为自己一直没接触过过内网这块的渗透，于是最近心血来潮，打算学习并总结一下关于内网渗透的知识，弥补自己后渗透这一块的不足

一、Lcx内网穿透之Windows版

lcx.exe是一个端口转发工具，有Windows版和Linux版两个版本，Windows版是lcx.exe,Linux版为portmap。

lcx有两大功能：

  • 端口转发(listen和slave成对使用）
  • 端口映射(tran)

然而强大的Lcx属于古董级别的转发工具了，会经常遭到拦截，不过还是总结一下原理和用法，以示尊敬：

1.端口转发

假设攻击的场景如上图：

1、 网络环境：
（1）主机A可以随意连接外网
（2）主机B在外网无法连接未经防火墙许可的端口，此处假设为3389端口

（ Windows 2000/2003 Server 远程桌面的服务端口 ）

2、原理：
（1）内网主机A主动连接外网主机B的80端口
（2）主机B的80端口接受来自主机A的数据并转到5678端口（此端口可以使用任意服务不冲突的端口）
（3）从5678端口输入的主机B数据也可以转到80端口
（4）转到80后相当于主机B由80端口输入数据，并传输到主机A
（5）主机A上的端口转发工具LCX再把从主机B的80 端口收到的数据转到主机A的3389端口。
（6）同理从主机A的3389端口输入的数据也可以到达主机B
